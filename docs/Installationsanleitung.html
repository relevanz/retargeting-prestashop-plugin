Installationsanleitung für Prestashop

Hinweis: Die folgende Installationsanleitung richtet sich an Shopbetreiber, die das releva.nz Plugin selbst installieren möchten.

Modul installieren

  1. Kopieren Sie mit Ihrem FTP-Programm das Verzeichnis relevanz in das Verzeichnis modules von Ihrem Shop.

Modul aktivieren

  1. Loggen Sie sich in den Shop Admin Ihres Webshops ein.
  2. Öffnen Sie im Menü unter "Module" den "Modul-Katalog".
  3. Suchen Sie mit Hilfe der Suchfunktion nach "releva".
  4. Klicken Sie Auf die Schaltfläche "Installieren" vom "releva.nz" Plugin.
  5. Das Modul befindet sich nun in der Navigation unter "releva.nz".

Modul einrichten

  1. Hinterlegen Sie in der Konfiguration vom releva.nz Plugin in dem entsprechenden Eingabefeld den von uns bereitgestellten API-Key.
  2. Klicken Sie auf die Schaltfläche "Speichern".
  3. Wenn der API-Key korrekt ist, wird Ihnen eine entsprechende Erfolgsmeldung angezeigt. Falls nicht, überprüfen Sie bitte, ob Sie den API-Key korrekt eingegeben haben.
  4. Die Einrichtung ist nun erfolgreich abgeschloßen. Sie haben nun Zugriff auf Statistiken und die Konfiguration ihres Werbebudgets unter dem Menüpunkt "releva.nz" → "Statistik".
